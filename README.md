# 문자열 덧셈 계산기

----

## 기능 구현 목록

### 1) 입력 및 선언
- [x] `Console.readLine()`으로 한 줄 입력을 읽는다.
- [ ] 입력이 비었으면 합을 0으로 처리한다.
- [ ] 기본 구분자 `,`, `:`를 준비한다.
- [ ] 커스텀 선언 `"//{delim}\n..."`을 인식하고 `{delim}`(한 글자)을 구분자에 추가한다.

### 2) 문자열 처리
- [ ] 모든 구분자로 문자열을 분리한다.
- [ ] 유효한 정수 토큰을 합한다.

### 3) 출력
- [x] 결과를 형식에 알맞게 `결과 : {합}` 으로 출력한다.

### 4)예외 처리
- [ ] `//`로 시작했는데 `\n`이 없거나 `{delim}`가 비었거나 길이가 1이 아닌 경우
- [ ] 연속 구분자(`"1,,2"`), 시작/끝이 구분자(`",1"`/`"1,"`) 등으로 숫자 없이 비어 있는 경우, 0으로 처리
- [ ] 정수와 구분자 외의 문자를 포함한 경우
- [ ] 음수 혹은 0을 입력한 경우
